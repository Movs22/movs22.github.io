Criei esta página como alternativa à aplicação oficial e ao Google Maps, bem como aos outros city-mappers que, ou apresentam anúcios ou colocam algumas funcionalidades por trás de um plano premium.


Como funciona?

Este site é hospedado no Github, através do Github pages.
Utiliza a API pública da CMetropolitana e os cerca de 2 meses de experiência que adquiri ao experimentar com a API pública (através de um bot no discord e outros projetos de uso pessoal).

Assim, este site não só permite ver as próximas partidas, como também o tipo de veículo (algo que não é partilhado em mais nenhuma plataforma) e, caso a API dos horários em tempo real vá abaixo (como têm acontecido ultimamente), consegue ainda estimar as chegadas através da posição dos veículos.

Para além disso, este site também apresenta algumas informações dos veículos (e.x. se têm uma porta avariada...), através de contribuições dos utilizadores deste site.


Como é que surgiu esta ideia?

Este projeto começou como um bot do discord que via quando é que os autocarros começavam/terminavam o serviço, com o objetivo de ver se, de facto, as linhas com menos serviços (e.x. a 1122) de facto circulavam. Logo aqui, tive de lidar com problemas da API (pois havia veículos que "desapareciam" por uns minutos e o bot assumia que tinham terminado o serviço).

Este bot evoluíu, passando a enviar avisos quando um autocarro que apanho diáriamente chegava à paragem X. A partir desta funcionalidade, fui expandindo o bot, criado um comando que me mostrasse as informações de qualquer paragem.
(Descobri também nesta altura que os serviços rodoviários de substituição da CP estão, internamente, registados como sendo parte da Linha 1998.)

A partir daqui, adicionei outro comando que me permitia ver todos os veículos a circular numa determinada linha e foi aqui que comecei a associar o ID do veículo (o número preto/branco que se encontra na parte da frente/de trás do autocarro) com o tipo de veículo.

Este bot viria a tornar-se bastante útil, tanto para mim como para uns amigos meus e, por isso, decidi criar este site, disponibilizando assim os meus conhecimentos da API da CMetropolitana para qualquer pessoa que possa vir a precisar.


Este projeto pertence à CMetropolitana?

Não, este projeto é completamente independente da CMetropolitana e da TML, utilizando apenas a API pública disponibilizada pela CMetropolitana e os meus conhecimentos dos serviços.


Porque é que este site só funciona na Z1?

Tecnicamente, este site poderia funcionar com as 4 zonas da CMetropolitana, contudo, dado que eu só conheço os serviços (e autocarros) da Zona 1, optei por excluir as outras 3 zonas deste site.
Para além disso, as outras 3 zonas têm veículos completamente diferentes, e são operadas por outros operadores com os quais não estou familiarizado (em vez da Viação Alvorada, que resultou da junção da Vimeca/LT, dos autocarros municipais de Oeiras e da Scotturb).

Nas paragens em que há autocarros das duas zonas (e.x. em Lisboa ou na Ericeira), optei por mostrar os veículos das outras zonas com uma cor diferente (e sem informações do tipo de veículo/chegada prevista), embora 99% dos utilizadores deste site provavelmente nunca se vão deparar com estes casos extremos.
